# smallLeek
微信小程序smallLeek又名小韭菜，使用uni-app开发

## 功能
实时更新基金估算净值和计算收益
### smallLeek实例
![我的韭菜助手](./src/static/QRcode/smallLeek.png)
微信搜索小程序：我的韭菜助手
## 技术栈
> 1、vue
> 2、uni-app
> 3、微信小程序

## 启动小程序前端
通过 vue-cli 创建 uni-app 项目
### 环境安装
#### 全局安装vue-cli
 * npm install -g @vue/cli
#### 创建uni-app
 * vue create -p dcloudio/uni-preset-vue my-project
此时，会提示选择项目模板，选择默认模板
#### 运行
 * npm run dev:mp-weixin
